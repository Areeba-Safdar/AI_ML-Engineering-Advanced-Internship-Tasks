# 📂 Repository Contents

## ✅ Task 1: News Topic Classifier Using BERT

### Objective
Fine-tune the *BERT (bert-base-uncased)* transformer model on the *AG News Dataset* to classify news headlines into four categories:
- World
- Sports
- Business
- Sci/Tech

### Technologies Used
- Python
- Google Colab
- Hugging Face Transformers
- PyTorch
- Hugging Face Datasets
- Gradio
- Scikit-learn

### Methodology
- Loaded and explored the AG News dataset.
- Preprocessed and tokenized text using the BERT tokenizer.
- Fine-tuned the pre-trained BERT model.
- Evaluated the model using Accuracy and F1-score.
- Built a Gradio interface for real-time news classification.

### Results
- *Accuracy:* ~92%
- *F1-Score:* ~92%

## ✅ Task 2: End-to-End ML Pipeline for Customer Churn Prediction

### Objective
Develop a reusable and production-ready machine learning pipeline to predict customer churn using the *Telco Customer Churn Dataset*.

### Technologies Used
- Python
- Google Colab
- Scikit-learn
- Pandas
- NumPy
- Joblib
- Matplotlib

### Methodology
- Loaded and cleaned the Telco Customer Churn dataset.
- Performed preprocessing using Scikit-learn Pipeline API.
- Trained Logistic Regression and Random Forest models.
- Applied GridSearchCV for hyperparameter tuning.
- Evaluated model performance.
- Exported the best pipeline using Joblib.

### Features
- Data preprocessing pipeline
- Hyperparameter tuning
- Reusable ML pipeline
- Model export for production use
